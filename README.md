# AI-Generated vs Real Image Classification

University group project for the course *Introdução à Aprendizagem Automática* (University of Aveiro, 2025/26). A binary classifier that tells real photographs apart from AI-generated images at 32×32 resolution using the [CIFAKE](https://www.kaggle.com/datasets/birdy654/cifake-real-and-ai-generated-synthetic-images) dataset, with a custom CNN and a ResNet-18 trained from scratch. The trained detectors are then evaluated on the [Defactify](https://huggingface.co/datasets/Rajarshi-Roy-research/Defactify_Image_Dataset) dataset to test transfer across five other generators.
